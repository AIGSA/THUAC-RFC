房间系统交互文档

概要

该文档定义了前端与测评端的的房间系统交互，原则上所有通信采用 Websocket 协议。

测评端地址：ws://JUDGER_IP:JUDGER_PORT/ws，该地址应由测评端初始化时提供给网站后端。

前端 -> 测评端

获得房间列表

{
    'type': 0
}

测评端 -> 前端

推送房间列表

当房间列表更新或者前端主动请求更新房间列表时，服务器应推送完整房间列表

{
    'type': 0
    'data': {
        'rooms': [
            {
                'room_id': 1000,
                'game': {
                    'id': 0,
                    'name': 'uno',
                    'alias': 'UNO 牌'
                }
                'players': [
                    {
                       'username': 'admin',
                       'email': 'admin@saiblo.com'
                    }
                ]
            }
        ]
    }
}
